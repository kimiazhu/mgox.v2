
## mgox

mgox.v2 is a friendly wapper of [mgo.v2](http://gopkg.in/mgo.v2).

this project is a custom version of [mgox](http://github.com/yaosxi/mgox)

## Installation

    go get github.com/kimiazhu/mgox.v2

## Features

* Insert
* Remove
* Set/Inc/Push/Pull/Replace
* Find
* Get
